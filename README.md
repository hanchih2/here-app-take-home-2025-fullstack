# Instruction

clone the repository 

```
git clone 
cd here-app-take-home-2025-fullstack
```

## Run the Server

install dependencies

```
cd server
npm install
```

the server should run on port 4000

```
npm start
```

## Run the Client

install dependencies

```
cd .. // if you are in server directory
cd client
npm install
```

start the client

```
npm start
```
